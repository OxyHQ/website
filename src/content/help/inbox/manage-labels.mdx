Labels are color-coded tags you can apply to any message. Unlike folders, a single message can carry multiple labels — so a receipt can be both "Work" and "Reimburse" without sitting in two places.

## Create a label

<Steps>
  <Step title="Open Settings → Labels">
    Tap **Manage labels** at the bottom of the sidebar.
  </Step>
  <Step title="Tap New label">
    Pick a name and a color.
  </Step>
  <Step title="Optional: add filters">
    Auto-apply this label to incoming messages that match your conditions (sender, subject, body keywords).
  </Step>
</Steps>

## Apply a label

Open any message and tap the tag icon — or drag the message onto the label in the sidebar.

## Smart labels

Some labels are computed automatically:

- **Receipts** — anything that looks like a purchase confirmation
- **Travel** — flights, hotels, ride-share confirmations
- **Newsletters** — list-managed mass mail

Smart labels never modify the message itself, just the way Inbox displays it.

<Callout type="tip">
  Use **Label → Archive on apply** if you want a label to auto-archive its messages. Great for newsletters you want to skim but not see in the main inbox.
</Callout>
