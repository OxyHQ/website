Ask Attio updates

We shipped Ask Attio to change how you work with your CRM.

This update takes it even further:

  • Ask Attio is now powered by Claude Sonnet 4.6 and Gemini 3.1 Pro.

  • Live prompt suggestions: Recap what you missed or clarify key points before a call ends.

  • Richer responses: Inline data tables and compact record links directly in chat.

  • Web source: Hover over the Searched web icon to see which sources informed an answer.

  • List queries: Search lists, check list membership, and filter by list attributes in Ask Attio.

  • Regenerate on demand: Use the Try again button to get a more detailed response, a more concise one, or give your own instructions.

  • Favorite chats: Pin Ask Attio conversations to jump back in instantly.

  • Personal Settings: Set Ask Attio's privacy behavior, choose a default model variant, and track your credit usage from Personal Settings.

Find out more.

Files on Deals and Custom objects

Keep contracts, decks, and documents where they belong — on the record.

The files tab is now available on all object types, including deals and custom objects.

Add the files tab via Configure Record Page, or find it included by default on new objects.

We've also refreshed the UI and rebuilt the backend for improved reliability.

Meet the Developer Platform

The Developer Platform is a modern, developer-friendly way to build apps, ship integrations, and automate work right where go-to-market happens.

This means you can:

  • Build with code: Sync and automate at scale with our REST APIs, or use our new App SDK to extend Attio with React and server side functions.

  • Start quick with low-code: Connect Attio with tools like Zapier, n8n, Gumloop, and Pipedream to automate fast.

  • Share what you build: Publish your app to the App Store for hundreds of thousands of Attio users to discover and install.

  • Join top builders: Teams like Slack, Linear, Granola, Pylon, and hundreds more are already building with the Developer Platform.
